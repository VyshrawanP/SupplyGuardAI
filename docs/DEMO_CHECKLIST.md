# Demo Checklist (Run-of-show)

## Before recording

- Use `.env` with `VITE_OFFLINE_MODE=true` for a frictionless run.
- Run `npm run lint` once.
- Run `npm run evaluate` and note 2–3 headline deltas to quote.

## During demo

1. Open `http://localhost:3000`
2. Toggle **Full view**
3. **Simulation**:
   - switch disaster modes
   - move sliders (show risk/route/mission changes)
4. **Operations**:
   - show live counters and notifications
5. **Impact**:
   - show replay-derived metrics
6. Close with SDG alignment + measurable outcomes
